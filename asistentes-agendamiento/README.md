# Introducción: Asistentes de Agendamiento en Vambe

Los **Asistentes de Agendamiento** en **Vambe** permiten automatizar por completo la gestión de citas mediante inteligencia artificial, funcionando 24/7 y sin intervención humana.

Este sistema se adapta tanto a **plataformas de reserva especializadas** como a **calendarios tradicionales**, según el tipo de negocio y operación que tengas.

Con Vambe, un cliente puede iniciar una conversación, resolver dudas, ver disponibilidad real, agendar una cita y recibir confirmaciones y recordatorios, todo de forma automática.

***

## Plataformas de agendamiento compatibles

En Vambe existen **dos grandes formas de agendar** con inteligencia artificial:

### Agendamiento con Calendarios (Google y Outlook)

Vambe se integra directamente con **Google Calendar** y **Outlook Calendar**, permitiendo gestionar la disponibilidad y las citas de uno o varios miembros del equipo.

Este tipo de agendamiento es ideal para:

* Reuniones comerciales
* Equipos de ventas
* Consultorías
* Negocios sin un sistema de reservas propio

Si quieres usar este tipo de agendamiento, revisa el artículo específico aquí:

Cómo agendar con Google Calendar y Outlook Calendar en Vambe\
https://academy.vambe.ai/v1/docs/c%C3%B3mo-iniciar-sesi%C3%B3n-y-vincular-google-calendar-en-vambe

***

### El poder de los calendarios con Vambe

Al conectar Google u Outlook Calendar con Vambe, se habilitan tres funciones clave:

* **Obtener disponibilidad**: detectar espacios libres reales en los calendarios
* **Crear evento**: agendar reuniones automáticamente
* **Cancelar evento**: eliminar citas cuando el cliente lo solicita

Esta integración permite coordinar agendas de **múltiples miembros del equipo**, evitando cruces de horarios y errores manuales.

{% stepper %}
{% step %}
### Obtención de disponibilidad inteligente

El sistema identifica los espacios libres más cercanos. Si no hay disponibilidad inmediata, propone automáticamente los próximos horarios disponibles.
{% endstep %}

{% step %}
### Definición de horarios laborales

Puedes configurar jornadas de trabajo específicas para evitar agendamientos fuera de horario.
{% endstep %}

{% step %}
### Múltiples calendarios conectados

Vambe puede consultar varios calendarios de agentes con permisos de administrador.

Si un agente no tiene disponibilidad pero otro sí, la IA gestiona correctamente la asignación.
{% endstep %}

{% step %}
### Métodos de asignación de citas

Puedes definir cómo se asignan las reuniones:

* Asignación aleatoria entre agentes disponibles
* Criterio personalizado con IA, según reglas del negocio (por sucursal, tipo de cliente, etc.)
* Asignación por agente específico, usando etiquetas o historial del cliente

Además, Vambe genera automáticamente la descripción del evento basada en la conversación y mensajes de confirmación enviados por WhatsApp.
{% endstep %}
{% endstepper %}

***

### Agendamiento con Aplicaciones de Reserva

El segundo tipo de agendamiento utiliza el bloque **Aplicaciones de Reserva**, diseñado para integrarse con plataformas especializadas como:

* **Dentalink / Medilink**
* **Reservo**
* **AgendaPro**

Este enfoque es ideal para clínicas, centros médicos y negocios que ya cuentan con un sistema de reservas estructurado.

Con estas integraciones, Vambe puede:

* Detectar si el paciente ya existe o crear uno nuevo
* Obtener disponibilidad real de profesionales
* Crear, reagendar o cancelar reservas automáticamente
* Mantener sincronizada toda la información del cliente

Para utilizar este tipo de agendamiento, es necesario configurar el bloque **Aplicaciones de Reserva** dentro del asistente.

En las siguientes secciones encontrarás el paso a paso para integrar cada plataforma.

***

## ¿Qué puedes lograr con los asistentes de agendamiento?

Con cualquiera de las dos modalidades (Calendarios o Aplicaciones de Reserva), Vambe te permite:

* Atender clientes 24/7
* Automatizar agendamientos y confirmaciones
* Reducir carga operativa del equipo
* Evitar errores manuales
* Mantener toda la información sincronizada

Además, puedes automatizar procesos adicionales como:

* Solicitud de documentos
* Confirmaciones previas
* Recordatorios automáticos
* Derivaciones a atención humana cuando sea necesario
