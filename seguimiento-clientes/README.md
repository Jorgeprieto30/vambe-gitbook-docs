# Seguimiento de Clientes: Fidelización y Retargeting

El ciclo de vida del cliente no termina cuando se cierra el chat. La sección de Seguimiento está diseñada para ayudarte a retener usuarios, medir la calidad de tu atención y reactivar conversaciones antiguas mediante automatizaciones avanzadas.

Aquí encontrarás las herramientas para conectar Vambe con tus propios sistemas (ERPs, CRMs externos) y estrategias para medir la satisfacción de tus usuarios.
