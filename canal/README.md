# Canales y Automatización

Bienvenido al centro de control de comunicaciones de Vambe. En esta sección configurarás los "puentes" que conectan tu empresa con tus clientes y las reglas automáticas que gestionarán esas interacciones.

Aquí encontrarás todo lo necesario para conectar tu mensajería (WhatsApp, Instagram, Facebook), configurar los permisos de Meta y definir cómo tu IA responderá a los comentarios públicos en tus posts.

***

#### 1. Ecosistema WhatsApp: ¿Cuál elegir?

En Vambe ofrecemos tres tipos de conexión para WhatsApp. Es vital entender sus diferencias antes de conectar:

* ☁️ **WhatsApp API (Oficial)**: La conexión más robusta. Ideal para empresas medianas/grandes. No usa teléfono físico, permite campañas masivas seguras y requiere plantillas de pago.
* 📱 **WhatsApp Dual (Híbrido)**: Lo mejor de dos mundos. Usas la API oficial (seguridad y automatización) pero mantienes la App de WhatsApp en tu celular para contestar manualmente.
* 🤳 **WhatsApp QR:** Conexión básica tipo "WhatsApp Web". Depende de tu teléfono encendido. Útil para operaciones pequeñas sin campañas masivas.

***

#### 2. Automatización de Comentarios (Instagram)

No solo respondas mensajes privados; domina tu presencia pública. Vambe te permite transformar cada comentario en un post en una oportunidad de venta.

Dispones de 4 estrategias de automatización:

1. **Global + IA**: Tu asistente virtual responde a todo (posts viejos y nuevos) con inteligencia artificial, derivando ventas al privado.
2. **Futura**: Dejas programadas respuestas fijas para un lanzamiento que harás próximamente.
3. **Post Individual**: Creas reglas específicas para una publicación viral o un concurso antiguo.
4. **Global (Clásica)**: Reglas fijas de limpieza y moderación para toda la cuenta.

***

#### 3. Configuraciones de Meta y Herramientas

Para operar profesionalmente, necesitas configurar tu entorno en Meta (Facebook) y utilizar herramientas de mensajería masiva:

* **Facturación**: Verificar tu negocio y agregar métodos de pago en Meta es obligatorio para usar la API oficial.
* **Plantillas**: Mensajes pre-aprobados necesarios para iniciar conversaciones o reactivarlas tras 24 horas.
* **Campañas**: El motor para enviar mensajes masivos a tus bases de datos.
