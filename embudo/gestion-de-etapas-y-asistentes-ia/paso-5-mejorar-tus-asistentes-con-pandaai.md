# Paso 5: Mejorar tus asistentes con PandaAI

{% stepper %}
{% step %}
### ¿Dónde se encuentra PandaAI?

PandaAI está disponible en todo momento en la **esquina superior derecha** de Vambe.

![](<../.gitbook/assets/image png Dec 24 2025 01 21 12 3653 PM.png>)

Desde ahí puedes abrirlo para:

* Resolver preguntas generales sobre asistentes.
* Obtener ayuda sobre flujos y configuraciones.
* Buscar soluciones rápidas a errores o dudas comunes.
* Recibir sugerencias para mejorar tus prompts.

PandaAI está conectado a la información estructurada de tus asistentes y embudos, por lo que es capaz de contextualizar sus respuestas.
{% endstep %}

{% step %}
### ¿Para qué sirve PandaAI?

PandaAI te puede ayudar en múltiples tareas, entre ellas:

* Comprender cómo funcionan los asistentes\
  Puedes pedirle explicaciones sobre bloques, funciones, acciones automáticas, metadatos, estructura del asistente, etc.
* Crear o mejorar prompts\
  PandaAI puede ayudarte a redactar instrucciones más claras, completas y consistentes con las buenas prácticas de Vambe.
* Encontrar errores en un asistente\
  Si un flujo no funciona, PandaAI puede detectar:
  * Instrucciones contradictorias
  * Faltas de formato
  * Falta de pasos clave
  * Problemas con funciones
  * Bloques mal estructurados
* Obtener ejemplos o plantillas\
  Si necesitas modelos para:
  * Pasos a seguir
  * Casos posibles
  * Bloques de no hacer
  * Funciones de cambio de etapa
  * Vector Based Search
  * Mensajes del asistente

PandaAI puede generarlos según tu caso.
{% endstep %}

{% step %}
### Uso de PandaAI dentro de cada bloque del asistente

Cada vez que edites un bloque —ya sea de identidad, instrucciones o información— verás un botón llamado:

**“Potenciar con PandaAI”**

Al hacer clic:

* PandaAI revisará el contenido del bloque.
* Detectará oportunidades de mejora.
* Sugerirá ajustes para claridad, coherencia y efectividad.
* Evitará contradicciones entre bloques.
* Propondrá correcciones basadas en buenas prácticas de Vambe.

Esto permite optimizar cada componente del asistente sin necesidad de rehacerlo desde cero.
{% endstep %}

{% step %}
### Usar PandaAI para corregir información o validar datos

Si necesitas verificar si la información que estás cargando es correcta o consistente, puedes usar PandaAI directamente.

Ejemplos de uso:

* “Corrige este texto y déjalo más claro.”
* “¿De dónde sacaste esta información?”
* “Ayúdame a estructurar esta tabla.”
* “¿Ves algún error en esta instrucción?”

PandaAI puede leer información que le entregues, mejorarla y ayudarte a dejarla lista para colocarla en un bloque correspondiente.

Enlace relacionado: [PandaAI](https://academy.vambe.ai/v1/docs/c%C3%B3mo-corregir-a-la-ia-cuando-se-equivoca-en-una-conversaci%C3%B3n)
{% endstep %}

{% step %}
### ¿Por qué PandaAI es importante?

Porque permite:

* Evitar errores antes de publicar un asistente.
* Acelerar el proceso de construcción.
* Mantener estándares de calidad consistentes.
* Resolver dudas sin necesidad de esperar soporte.
* Lograr asistentes más efectivos, claros y operativos.

PandaAI actúa como un copiloto dentro de Vambe, ayudando a que cada usuario pueda construir asistentes sólidos incluso sin conocimiento técnico previo.
{% endstep %}
{% endstepper %}
